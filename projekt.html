Statistikprogram till Panduro

Kunden till detta projekt var butiksägare och ville ha bättre koll över butikernas statistik. Efter några dagars diskussion där vi identifierade problem och önskemål, kunde vi på Portfolio Group sätta oss ner för att skissa och koda detta system. Systemet innehöll tydlig statistik med grafer, staplar och cirkeldiagram som vi format utifrån kundens önskemål. Genom graferna kunde man bland annat utläsa butikens utgifter och inkomster varje månad, sålda produkter inom olika kategorier, antal kunder och en översikt över personalens arbetade timmar.

Kompetens: SQL - Java

Mobilapplikation för reklamblad

Vi på Portfolio Group har tagit fram en mobilapplikation efter en kunds önskemål. Vår kund hade identifierat ett hål i marknaden där många personer väljer att inte få hem reklam i brevlådan längre, samtidigt som det är omständigt att leta fram veckans bästa erbjudanden genom att gå igenom alla butikers hemsidor var för sig. Därför fick vi i uppdrag att skapa en app som samlar alla reklamblad från butiker som finns i din närhet. Appen har även en funktion där man som användare kan filtrera bort butiker man inte är intresserad av. Vi är otroligt glada och stolta över detta arbete som idag har tusentals användare dagligen.

Kompetens: C#

Webbplats till skidbyrån Skilink

Skilink arbetar intensivt för att öka intresset för vintersport samt för att skapa minnesvärda skidresor för sina gäster. Fokus ligger på att ge kunden den bästa upplevelsen från början till slut. I de flesta fall kommer Skilinks kunder i kontakt med företaget för första gången via deras webbplats. Redan här börjar resan mot kundnöjdhet och när de bestämde sig för att utveckla sin webbplats fick vi äran att skapa den nya. Denna viktiga plattform är länken mellan kunderna och medarbetarna och behöver vara användarvänlig, tydlig och informationsrik för kunderna samtidigt som anslutningen till företagets bokningssystem och ekonomisystem är stabil för att undvika ev. problem som kan dra ner kundnöjdheten.

Kompetens: HTML – CSS – JavaScript - jQuery

Logga och underhåll till nystartat företag

I detta projekt fick vi på Portfolio Group i uppgift av kunden att skapa en logga till ett nystartat privatföretag och sedan implementera loggan på företagets hemsida. Kundens vision och budskap för företaget diskuterades och klarställdes noga innan vi på Portfolio Group började vår kreativa process. Loggan skapades i verktyg som Photoshop och tillämpades sedan på företagets befintliga hemsida. Efter projektets slut fortsatte kommunikationen mellan kunden och vi på Portfolio Group för underhållning av förtagets hemsida och framtida system.

Kompetens: PhotoShop - Illustrator - JavaScript
